`/worldcup/community/create`. The bulk-import panel + a YouTube
playlist URL is the fastest path for the bigger sizes.

## Tips when filling

- For 4/8 sizes: hand-pick 1 winning track per artist (no album cuts)
- For 16/32: include some deep cuts so the bracket has surprises
- For 64+: use bulk-import from YT playlist (R23b)
- Tag with: genre + era + region (e.g. `k-pop, 2020s, idol`)
- Description: explain the THEME and what makes pairs interesting
- Random shuffle is on by default — each play randomises matchups
